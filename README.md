🍦 Baskin Robbins Sales Performance Dashboard (Power BI)

An interactive Power BI dashboard built to analyze retail sales performance, customer purchasing behavior, product trends, and regional revenue insights for Baskin Robbins.

🖼 Dashboard Preview
![Baskin Robbins Sales Dashboard](https://raw.githubusercontent.com/Rudrax03/BR-Sales-Performance-Dashboard/main/Snapshot%20of%20the%20BR%20Sales%20Performance%20Dashboard.jpeg)
🚀 Project Overview

This dashboard provides a comprehensive view of overall sales performance through dynamic KPIs and interactive visuals.

📌 Key Metrics

💰 Total Sales: 24.48M

📦 Total Quantity Sold: 150K

🛒 Total Orders: 50K

🧾 Average Bill Value: 489.56

🎁 Total Discount Given: 21.50K

📊 Dashboard Insights

🍨 Ice Cream category contributes the highest revenue

🏆 Karnataka leads in state-wise sales

⏰ Afternoon orders dominate sales volume

💳 UPI & digital payments form a major share of transactions

📈 Top 5 products drive a significant portion of total revenue

🔎 Features

Year & Month filtering

Category-wise analysis

State-wise performance comparison

Payment mode distribution

Time-of-day order analysis

Top products ranking

Interactive slicers & dynamic KPIs

🛠 Tech Stack

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (KPIs & Measures)

Data Modeling

🎯 Objective

To create a data-driven sales performance dashboard that helps analyze revenue trends, customer behavior, and regional performance using interactive business intelligence tools.
